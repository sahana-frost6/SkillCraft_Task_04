# Titanic Data Analysis & EDA (SkillCraft Technology Task 02)

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the classic Titanic dataset. The goal is to identify factors that influenced survival rates, such as socio-economic status (class), age, and gender, using data cleaning and visualization techniques.

## 📊 Key Insights
- **Gender Bias:** Females had a significantly higher survival rate compared to males, consistent with the "women and children first" protocol.
- **Class Impact:** Passengers in **1st Class** had the highest survival probability, while those in **3rd Class** faced the highest casualty rates.
- **Age Distribution:** Children and young infants showed higher survival peaks, while elderly passengers were among the most vulnerable.
- **Fare Correlation:** Higher fares (strongly linked to 1st class) correlate positively with survival, as shown in the correlation heatmap.

## 🛠️ Technologies Used
- **Python**
- **Pandas** (Data Cleaning & Analysis)
- **Seaborn & Matplotlib** (Statistical Visualizations)
- **NumPy** (Numerical Operations)

## 📂 Dataset Summary
The analysis was performed on the following key features:
- `Survived`: 0 = No, 1 = Yes
- `Pclass`: Ticket class (1, 2, 3)
- `Sex`: Gender
- `Age`: Age in years
- `Fare`: Passenger fare
- `Embarked`: Port of Embarkation
